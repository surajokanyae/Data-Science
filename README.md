# Data Science Portfolio

This repository contains my practical work and learning projects in **Data Science and Data Analysis**.  
It covers the full data pipeline — from data collection and cleaning to analysis, visualization, and basic machine learning.

The projects here reflect hands-on practice using industry-relevant tools and real-world datasets.

---

## Tools & Technologies

- **Excel** – Data cleaning, analysis, formulas, pivot tables, dashboards
- **SQL** – Data querying, joins, aggregations, subqueries
- **Power BI** – Interactive dashboards and data visualization
- **Python** – Data analysis, automation, and scripting
- **Machine Learning** – Basic predictive modeling and evaluation
