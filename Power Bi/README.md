# Power BI for Data Visualization

This folder contains my **Power BI projects** focused on data visualization, reporting, and business insights.  
The work here demonstrates how raw data is transformed into clear, interactive dashboards that support decision-making.

Power BI is used as a storytelling and reporting tool, not just for charts.

---

## Skills Demonstrated

- Data modeling and relationships
- Data transformation using Power Query
- Creating interactive dashboards
- Writing basic DAX measures
- Visual storytelling and insight communication

---

## Tools & Features Used

- Power BI Desktop
- Power Query (ETL process)
- DAX (CALCULATE, SUM, COUNT, basic measures)
- Filters, slicers, and drill-downs
- Interactive visuals and reports
- business_insights.pbix
- summary_report.pbix
